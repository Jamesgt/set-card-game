Welcome to the Set card game

How to play

There are 81 unique cards.

All card has 4 properties: shape, pattern, fill, number.

All properties have 3 possible values.

A set is only valid if all properties have the same value or none of the values match on the 3 card.

If a player thinks there is a Set, press SPACE.

A timer will start to help ensure that the player is pointing quickly the 3 cards.

Other players have to validate if that is really a Set or not.

If it is, the player scores a point. The cards in the Set are dismissed.

If not, then the player loses a point. You cannot go under 0.

If all players agree that there is no Set on the table, you can draw 3 more cards.
